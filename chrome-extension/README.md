# JobFill AI Chrome Extension

Auto-fill job applications with AI-powered field matching and stealth mode.

## Installation

1. Download this folder
2. Create icon images:
   - Go to https://favicon.io/emoji-favicons/high-voltage/
   - Download and rename to icon16.png, icon48.png, icon128.png
   - Place in the `icons/` folder
3. Open Chrome and go to `chrome://extensions`
4. Enable "Developer mode" (toggle in top right)
5. Click "Load unpacked"
6. Select this folder

## Features

- **Stealth Mode**: Human-like typing delays and mouse movements
- **AI Field Matching**: Intelligently matches your profile to form fields  
- **Multi-Platform**: LinkedIn, Indeed, Greenhouse, Lever, Workday, Glassdoor, ZipRecruiter
- **Application Tracking**: Logs all applications to your dashboard
